# Exercício de Simulação de Phishing

## Visão Geral
Este repositório contém um exercício educacional criado durante um treinamento de segurança cibernética (usando a ferramenta SET - Social Engineering Toolkit). O objetivo foi entender conceitos básicos de phishing em um ambiente controlado e isolado, para fins de aprendizado em pentesting ético.

## O Que Foi Feito
- Configurei um ambiente de teste local com máquinas virtuais isoladas.
- Usei o SET para simular um cenário de phishing básico, gerando páginas falsas e coletando logs de teste (sem dados reais ou alvos externos).
- Testei detecção de vulnerabilidades simuladas, como credenciais fictícias, para analisar respostas de defesa.
- Incluí screenshots do processo (veja pasta `screenshots/`).

Tudo rodou em rede privada, sem conexão à internet ou impacto externo. Foi um exercício pessoal para praticar conceitos de cibersegurança.

## Aviso Ético
- **Apenas para estudo**: Não use em produção ou sem autorização explícita. Phishing real é ilegal no Brasil (Lei 14.155/2021 e Marco Civil da Internet).
- Ambiente: Kali Linux + VMs isoladas.
- Se tiver dúvidas, pergunte nos Issues – explico o contexto educacional.

## Requisitos
- Kali Linux ou similar.
- SET instalado (`sudo apt install set`).
